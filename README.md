### Overview

A fork of [Dracula Official](https://marketplace.visualstudio.com/items?itemName=dracula-theme.theme-dracula), Illusion is a dark theme for Visual Studio Code.

---

### Install

<ol>
  <li>Go to View -> Command Palette or press ⌘+shift+P</li>
  <li>Then enter Install Extension</li>
  <li>Write Illusion</li>
  <li>Select it or press Enter to install</li>
</ol>
